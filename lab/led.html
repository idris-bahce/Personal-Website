Overview

In Raspberry Pi projects, LEDs are typically used as indicator lights, illumination devices, display screens, etc., offering advantages such as low power consumption, long lifespan, and quick response.

Components

  • Raspberry Pi x 1
  • T-Extension Board x 1

  • 40-Pin Cable x 1

Build Process

Step 1: Hardware Assembly

Connect an LED to the Raspberry Pi by wiring GPIO17 (pin 11) → resistor (220 Ω) → LED anode (long leg). Connect the LED cathode (short leg) to GND (pin 6).

Step 2: Programming

Compile and run the C program on the Raspberry Pi to control the LED.

Code

Here's the complete Raspberry Pi sketch for LED blinking:

#include <stdio.h>
#include <unistd.h>   // sleep()
#include <lgpio.h>   // GPIO API

int main(void) {
    int h = lgGpiochipOpen(0);      // open /dev/gpiochip0
    int led = 17;                   // BCM numbering

    // Claim GPIO17 as output, initial level LOW
    lgGpioClaimOutput(h, 0, led, 0);

    for (int i = 0; i < 10; i++) {
        lgGpioWrite(h, led, 1);     // drive pin HIGH (≈3.3 V) → LED on
        sleep(1);
        lgGpioWrite(h, led, 0);     // drive pin LOW (0 V) → LED off
        sleep(1);
    }

    lgGpiochipClose(h);             // release the chip
    return 0;
}

Results

The LED blink project successfully demonstrates basic GPIO control on the Raspberry Pi. The LED connected to GPIO17 turns on and off at one-second intervals, confirming correct hardware wiring and program execution. This provides a foundational step for building more advanced hardware-interfacing projects in C.
