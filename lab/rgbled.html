RGB LED

Oktober 2025

← Back to Lab

Overview

In this project, we control an RGB LED using the Raspberry Pi and the lgpio library in C. By combining the red, green, and blue channels, we can create different colors (red, green, blue, yellow, cyan, magenta, white). This project builds on the basic LED blink and introduces multi-pin coordination for color mixing.

Components

  • Raspberry Pi x 1
  • T-Extension Board x 1
  • RGBLED x 1
  • 40-Pin Cable x 1
  • Breadboard & Jumper Wires
  • 220Ω Resistor x 3

Build Process

Step 1: Hardware Assembly

Identify the RGB LED type:

  • Common cathode → long pin to GND.
  • Common anode → long pin to 3.3 V.

Connect each LED leg through a 220 Ω resistor:

  • Red → GPIO17 (pin 11)
  • Green → GPIO27 (pin 13)
  • Blue → GPIO22 (pin 15)
  • Common pin → GND (cathode) or 3.3 V (anode).

Step 2: Programming

Compile and run the C program on the Raspberry Pi to control the LED.

Code

Here's the complete Raspberry Pi sketch for LED blinking:

#include <stdio.h>
#include <unistd.h>   // sleep()
#include <lgpio.h>   // GPIO API
#define R_PIN 17
#define G_PIN 27
#define B_PIN 22

// helper: set RGB states
static void setRGB(int h, int r, int g, int b) {
    lgGpioWrite(h, R_PIN, r);
    lgGpioWrite(h, G_PIN, g);
    lgGpioWrite(h, B_PIN, b);
}

int main(void) {
    int h = lgGpiochipOpen(0);
    if (h < 0) { perror("lgGpiochipOpen"); return 1; }

    if (lgGpioClaimOutput(h, 0, R_PIN, 0) < 0 ||
        lgGpioClaimOutput(h, 0, G_PIN, 0) < 0 ||
        lgGpioClaimOutput(h, 0, B_PIN, 0) < 0) {
        perror("lgGpioClaimOutput");
        lgGpiochipClose(h);
        return 1;
    }

    // Cycle through colors
    for (int i = 0; i < 5; i++) {
        setRGB(h, 1, 0, 0); sleep(1); // Red
        setRGB(h, 0, 1, 0); sleep(1); // Green
        setRGB(h, 0, 0, 1); sleep(1); // Blue
        setRGB(h, 1, 1, 0); sleep(1); // Yellow
        setRGB(h, 0, 1, 1); sleep(1); // Cyan
        setRGB(h, 1, 0, 1); sleep(1); // Magenta
        setRGB(h, 1, 1, 1); sleep(1); // White
        setRGB(h, 0, 0, 0); sleep(1); // Off
    }

    // ensure off before exit
    setRGB(h, 0, 0, 0);
    lgGpiochipClose(h);
    return 0;
}

Results

The RGB LED successfully cycles through red, green, blue, yellow, cyan, magenta, and white, demonstrating additive color mixing. This project shows how multiple GPIO pins can be coordinated to create visual effects, building the foundation for more advanced projects like PWM-based color fading or LED displays.
